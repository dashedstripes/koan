# Text

```css

/* Font Size */
.fz-{breakpoint}-{font size}
.fz-{sm (small) | md (medium) | lg (large) | blank)-{025 | 05 | 075 | 1 | 125 | 15 | 175 | 2-8 | i (initial)}

.fz-4 === font-size: 4rem
.fz-sm-4 === font-size: 4rem /* (only on small screens) */

/* Font Weight */
.fw-{b (bold) | n (normal) | l (lighter)}

.fw-b === font-weight: bold


/* Font Style */
.fs-{i (italic) | n (normal) | o (oblique)}

.fs-i === font-style: italic


/* Text Decoration */
.td-{n (normal) | u (underline) | lt (line-through)}

.td-u === text-decoration: underline


/* Text Transform */
.tt-{u (uppercase) | l (lowercase) | c (capitalize)}

.tt-u === text-transform: uppercase

/* Text Align */
.ta-{l (left) | r (right) | c (center) | j (justify) | i (initial)}

.ta-c === text-align: center

/* Font Families */
.ff-{sans-serif | serif | custom}

.ff-serif === font-family: serif

```
