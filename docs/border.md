# Border

```css

/* Border Style */
.bs-{n (none) | h (hidden) | d (dotted) | da (dashed) | s (solid) | do (double) | g (groove) | r (ridge) | i (initial)}

.bs-d === border-style: dotted

/* Border Width */
.b{t (top) | b (bottom) | l (left) | r (right | blank)}w-{1-10}

.bw-6 === border-width: 6px
.btw-4 === border-top-width: 4px

/* Border Color */
.bdrc-{color}

.bdrc-red === border-color: $red

/* Border Radius */
.br-{1-10}

.br-6 === border-radius: 6px

```
