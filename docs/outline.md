# Outline

```css

/* Outline Style */
.os-{n (none) | h (hidden) | d (dotted) | da (dashed) | s (solid) | do (double) | g (groove) | r (ridge) | i (initial)}

.os-d === outline-style: dotted

/* Outline Width */
.ow-{1-10}

.ow-6 === outline-width: 6px

/* Outline Color */
.oc-{color}

.oc-red === outline-color: $red
```
