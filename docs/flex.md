# Flex

```css

/* Flex Wraps */
.fw-{w (wrap) | nw (nowrap) | wr (wrap-reverse) | i (initial)}

.fw-w === flex-wrap: wrap

/* Justify Content */
.jc-{fs (flex-start) | fe (flex-end) | c (center) | sb (space-between) | sa (space-around) | i (initial)}

.jc-sb === justify-content: space-between

/* Align Items */
.ai-{s (stretch) | c (center) | fs (flex-start) | fe (flex-end) | b (baseline) | i (initial)}

.ai-c === align-items: center

/* Flex Directions */
.fd-{r (row) | rr (row-reverse) | c (column) | cr (column-reverse) | i (initial)}

.fd-rr === flex-direction: row-reverse

```
